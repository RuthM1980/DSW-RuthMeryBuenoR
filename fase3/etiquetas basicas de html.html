¿Qué es una etiquetas HTML?


HTML es un markup language, lo que significa que el texto en una página web está «marcado» con estos
códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.

Estas etiquetas de marcado son las propias etiquetas HTML.
Hay una serie de etiquetas que son las más usadas para crear cualquier documento HTML:

• Se utiliza body: para el contenido
• head: para información sobre el documento, es donde podremos indicar los metadatos:
título del documento, hojas de estilos
• div: nos ayuda a estructurar el documento en secciones.
• a: para enlaces; el atributo href, es donde ponemos el enlace al que queremos conectar.
Targe, sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.
• strong: para poner el texto en negrita
• br: para saltos de línea
• H1…H6: para títulos dentro del contenido
• img: para añadir imágenes al documento
• ol: para listas ordenadas, ul: para listas desordenadas, li: para elementos dentro de la lista
• p: para parágrafos
• span: para estilos de una parte del texto.
